# 序章-计算机视觉从入门到放弃

                计算机视觉是研究如何使电脑来看世间万物的科学。总的来说就是指用摄像头和计算机代替人眼对目标（object）进行识别（detection\)、跟踪（tracking）和测量（measearing）等。并进一步做处理成为更适合数据输出。
